## What Batch4LLM can do?

<CardGrid stagger>
	<Card title="Process file sets with LLMs" icon="rocket">
        You can setup a batch running huge number of files against an LLM with your custom prompt in minutes.
	</Card>
	<Card title="Connect to any LLM Provider" icon="star">
        All popular LLM providers are supported, including OpenAI, Gemini, Anthropic, DeepSeek and Mistral.
        And you can also configure a custom base url with the openai client.
	</Card>
	<Card title="Provider features" icon="puzzle">
		Provider specific features like file uploads, provider sided batches and structured outputs are fully supported.
	</Card>
    <Card title="Export as excel" icon="cloud-download">
        With just two clicks you can export your batches as an excel spreadsheet or csv.
        Containing either just your results or all the data you need for scientific evaluation.
    </Card>
</CardGrid>
